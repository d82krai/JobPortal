# QuikSync Job Aggregator Frontend

This is a **React + TypeScript + Vite** single-page application that provides a professional UI for the QuikSync Job Aggregator & ATS platform.

It is designed to sit in front of the backend described in:

- `requirements.md`
- `er-diagram.md`
- `api-docs.md`
- `core-api.yaml`, `ats-api.yaml`, `comms-api.yaml`, `monetization-llm-api.yaml`

## Tech stack

- React 18 + TypeScript
- Vite
- MUI (Material UI) for design system and theming
- React Router for routing

## Getting started

```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
